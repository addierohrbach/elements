An introduction into block elements.

Block:

Positioning content using the "block" value adds an element on a new line, stacking one on top of another. It also will occupy the available width. Block-level elements can be nesting inside each other or wrap inline-level elements.

Block-level elements are most commonly used for larger pieces of content (ex: paragraphs, etc).

For example: headings and paragraphs are block-level elements because they take up the whole width of the page and move the next element below them.

Inline:

Positioning content using the "inline" value will cause the elements to line up one after the other, and only maintin the width of their content. They can be nested inside each other, but cannot wrap block-level elements.

Inline-level elements are most commonly used for smaller pieces of content (ex: few world, small images, etc).

For example:
The "bold" and "italicize" elements are inline-level elements because they fall into the normal flow of the document.

Inline Block:

Positioning content using the "inline-block" value is helpful when trying to lay out pages or place elements next to one another within a line.

This value will display elements within a line while allowing them to still retain all properties (ex: height, width, margin, etc).
